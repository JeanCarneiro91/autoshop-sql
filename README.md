# Projeto AutoShop – Banco de Dados (Experiência Prática IV)

Este repositório contém os scripts SQL do projeto AutoShop.

## Como usar no MySQL Workbench

1. Abra o MySQL Workbench.
2. Conecte na sua instância local.
3. Abra uma New Query Tab.
4. Execute os arquivos na ordem:

- 01_create_tables.sql  
- 02_insert.sql  
- 03_selects.sql  
- 04_updates_deletes.sql  

## Conteúdo

- Criação do banco e tabelas
- Inserção de dados
- Consultas usando SELECT, JOIN e ORDER BY
- Comandos de UPDATE e DELETE
